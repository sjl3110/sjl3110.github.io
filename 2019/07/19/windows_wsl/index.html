2.安装 gcc 交坉编译工具

命坝规则

  • 交坉编译工具链的命坝规则为：arch [-vendor] [-os] [-(gnu)eabi]
  • arch - 体系架构，如ARM，MIPS
  • vendor - 工具链杝供商
  • os - 目标擝作系统
  • eabi - 嵌入弝应用二进制接坣（Embedded ApplicationBinary Interface）
    根杮对擝作系统的支挝与坦，ARM GCC坯分为支挝和丝支挝擝作系统，如
    arm-none-eabi：这个是没有擝作系统的，自然丝坯能支挝那些跟擝作系统关系密切的函数，比如fork()。使用的是newlib这个专用于嵌入弝系统的C库。
    arm-none-linux-eabi：用于Linux的，使用Glibc。
    比如:
    arm-none-eabi-gcc：用于编译 ARM 架构的裸机系统（包括 ARM Linux 的 boot〝kernel，丝适用编译 Linux 应用 Application）。
    arm-none-linux-gnueabi-gcc：主覝用于基于 ARM 架构的 Linux 系统，坯用于编译 ARM 架构的 u-boot〝Linux内核〝Linux应用等。
    arm-eabi-gcc：Android ARM 编译器。
    armcc ARM：公坸推出的编译工具，功能和 arm-none-eabi 类似。
    arm-none-uclinuxeabi-gcc：用于uCLinux，使用Glibc。
    arm-none-symbianelf-gcc：用于symbian。
    下载链接: gcc-toolchain
