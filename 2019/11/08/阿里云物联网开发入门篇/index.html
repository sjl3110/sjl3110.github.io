阿里云物联网开发入门篇

发表于 2019-11-08 | 分类于 物联网

电路连接

  • Arduino在使用串口连接时，不建议使用RX0和TX0，使用其他的数据口作为RX和TX，这里选择RX=D13，TX=D12。

  • ESP8266有两个UART口，平常烧写使用的UART0，在模块中同时也配置为Micro USB接口。
    • 烧写时，使用转接线接USB连接电脑（注意不要连接分线器，会无限循环掉电上电，世雄解释电压不稳）。

  • 将UART0（RX TX G 3V3）连接Arduino（RX接TX，TX接RX，共地），便于Arduino将数据直接传输给ESP8266。（这里还有个小技巧，UART0与Arduino连接时，电脑端连接Micro USB也可以获得数据，反正只要两边不同时向ESP8266发送数据就不会出现BUG，这样也可以在电脑端看到运行日志）

  • 将UART1（D4=TX G）通过串口转USB连接电脑，注意UART1只有TX引脚，也就是只能向电脑发送数据，这里我们用作打印一些数据，例如将UART0获取的Arduino传感器数据发送给电脑，检查是否正确。
    • BME280使用I2C接口，SCL和SDA对应连接即可，还有3V3供电及共地。

简单说明

文件结构


从上依次往下：

  • /includes：C标准头文件，从cygwin中引入；
  • /app：编译的文件夹；
    • /include：一些头文件，比如/driver/uart\mqtt、md5加密等;
    • /mqtt：mqtt的相关函数
    • /user：
      • uaer_main.c为主要文件；
      • uart.c、cJSON.c、user_wifi.c为一些函数文件；
      • Makefile为编译文件；
    • Makefile为编译文件；
  • /bin：编译生成的固件存储位置；
  • /include：8266提供的API头文件；
  • /lib：API头文件的库；
